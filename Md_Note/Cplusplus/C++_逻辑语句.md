# C++_逻辑语句

1. `if...else`语句

   ```C++
   if(condition){
       //statements;
   } else {
       //statements;
   }
   ```

2. `while` 语句：当条件满足时，`while`里的语句才会被执行

   ```C++
   while(condition){
       //statements;
   }
   ```

3. `do...while`语句：`do`内的语句将至少执行一次

   ```c++
   do{
       //statements;
   } while (condition);
   ```
